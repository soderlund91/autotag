Home Screen Companion — Setup Guide

All features at a glance
rocket_launch Quick Start
1
Enter API Keys — go to the Settings tab.
Trakt: trakt.tv/oauth/applications → New App → set Redirect URI to http://localhost. Copy the Client ID.
MDBList: mdblist.com/preferences → scroll to "API Access".
2
Add a Source — open the Tag & Collection tab, click + Add New Source.
3
Configure — give it a tag name, paste a list URL or pick a local source, and optionally enable a collection.
4
Save & Sync — click Save Settings, then use the Run button → Full sync. Done!
input Source Types
public External
Paste a list URL from Trakt or MDBList. The plugin fetches the items and tags matching titles in your library.
video_library Local
Pick an existing playlist or collection in Emby. Every item in that list gets the configured tag or collection.
tune MediaInfo
Filter your library by technical properties — codec, resolution, HDR, audio format, genre, actors, ratings and more.
local_offer Tags & Collections
label
Tag — An Emby tag applied to every matched item. Automatically added when an item appears in the list and removed when it leaves.
collections_bookmark
Collection — Creates and maintains an Emby collection. Name, description, and poster are all customisable. Enable Only Collection to skip tagging entirely.
block
Blacklist — Titles listed here are permanently excluded from this source, one per line.
workspace_premium
Priority Override — When this entry is in-schedule, all other entries sharing the same tag or collection are suppressed. Ideal for seasonal overrides (e.g. "Spotlight Halloween" takes over "Spotlight").
calendar_month Scheduling
all_inclusive
No schedule — Entry is always active.
date_range
Specific dates — Active between a start date and an end date.
repeat
Recurring — Activate on specific days every week, month, or year. Multiple intervals can be stacked on a single entry.
analytics MediaInfo Filters
Build filter rules from four property categories. Rules within a group use AND/OR logic; the NOT toggle inverts any rule. Numeric properties support =  >  ≥  <  ≤.
Video
Resolution (4K, 8K, HD…), Video Codec (HEVC, H264…), HDR type (HDR10, Dolby Vision…)
Audio
Format (Atmos, DTS-HD MA, TrueHD…), Channels (stereo, surround…)
Content
Genre, Actor, Studio, IMDB ID (comma-separated list supported)
Metrics
Community Rating, Year, Runtime (in minutes)
home Home Screen Companion
manage_accounts
Copies the home screen section layout from a source user to one or more target users. Useful for keeping multiple profiles in sync.
sync
Enable automatic sync (runs alongside the scheduled task), or trigger a manual sync from the Home Screen tab. Runs independently from the tag/collection sync.
