Home Screen Companion — Setup Guide

All features at a glance
rocket_launch Quick Start
1
Enter API Keys — go to the Settings tab.
Trakt: trakt.tv/oauth/applications → New App → set Redirect URI to http://localhost. Copy the Client ID.
MDBList: mdblist.com/preferences → scroll to "API Access".
2
Add a Source — open the Tag & Collection tab, click + Add New Source.
3
Configure — use the tabs inside each source: Source (pick type & URLs), Tag (tag name), Collection (optional collection), Schedule (optional date rules).
4
Save & Sync — click Save Settings, then use the Run button → Full sync. Done!
input Source Types
public External
Paste a list URL from Trakt or MDBList. The plugin fetches the items and tags matching titles in your library. Multiple URLs per source are supported.
video_library Local
Pick an existing playlist or collection in Emby. Every item in that list gets the configured tag or collection.
tune MediaInfo
Filter your library by technical properties — codec, resolution, HDR, audio format, genre, actors, ratings, and user activity (watched status, last played, play count).
tab Source Settings
toggle_on
Active toggle — The checkbox in the source header enables or disables the source entirely without deleting it. Disabled sources are skipped during sync.
badge
Display Name — A friendly label shown in the source header for your own reference. Separate from the Tag Name used in Emby.
input Source tab
Choose the source type and add URLs or local sources. Each entry has its own Max items limit (0 = unlimited).
label Tag tab
Toggle tagging on or off. Set the Tag Name that will be applied to matched items in Emby.
collections_bookmark Collection tab
Optionally create an Emby collection. Set a name, description, and poster. Enable Only Collection to skip tagging entirely.
calendar_month Schedule tab
Define when this source is active — specific date ranges or recurring rules. Add a Priority Override to suppress other sources sharing the same tag when this one is active.
block Blacklist tab
IMDB IDs listed here are permanently excluded from this source, regardless of what the list says.
tv Home Screen tab
Automatically add this source's items as a home screen section for specific users each time sync runs.
local_offer Tags & Collections
label
Tag — An Emby tag applied to every matched item. Automatically added when an item appears in the list and removed when it leaves.
collections_bookmark
Collection — Creates and maintains an Emby collection. Items are added and removed automatically on every sync.
check_box_outline_blank
Only Collection — Skip tagging entirely and only maintain the collection. Useful when you want a clean collection without adding tags to individual items.
image
Collection Poster — Upload a local image file or paste a URL to set a custom poster for the collection. Stored on the server and refreshed on every sync.
tv
Home Section per source — Each source can add its matched items as a dedicated home screen section for specific users. Configure this in the Home Screen tab inside each source card. Independent from the Home Screen Companion copy feature.
block
Blacklist — IMDB IDs listed in the Blacklist tab are permanently excluded from this source.
workspace_premium
Priority Override — When this entry is in-schedule, all other entries sharing the same tag or collection are suppressed. Ideal for seasonal overrides (e.g. "Spotlight Halloween" takes over "Spotlight").
calendar_month Scheduling
all_inclusive
No schedule — Entry is always active.
date_range
Specific dates — Active between a start date and an end date.
repeat
Recurring — Activate on specific days every week, month, or year. Multiple intervals can be stacked on a single entry.
analytics MediaInfo Filters
Build filter rules from five property categories. Rules within a group use AND/OR logic; the NOT toggle inverts any rule. Numeric properties support =  >  ≥  <  ≤. Activity properties can be scoped to any user, all users, or a specific user.
Video
Resolution (4K, 8K, HD, SD), Video Codec (HEVC, H264, AV1…), HDR type (HDR10, Dolby Vision, HLG)
Audio
Format (Atmos, DTS-HD MA, TrueHD, AC3…), Channels (stereo, surround, mono), Audio Language
Content
Genre, Actor/Cast, Director, Writer, Studio, Title, Tag, Content Rating, IMDB ID (comma-separated list supported)
Metrics
Community Rating, Year, Runtime (minutes), Date Added (days ago), Date Modified (days ago), File Size (MB)
Activity
Watched / Unwatched (IsPlayed), Last Played (days ago), Play Count — scoped per user: any, all, or a specific user
home Home Screen Companion
The Home Screen tab has two sub-sections: Manage and Copy & Sync.
view_list
Manage — Create and maintain custom home screen sections. Each section is linked to a source and automatically kept up to date for the selected users when sync runs. Configure per-source home sections from the Home Screen tab inside each source card.
copy_all
Copy & Sync — Copy the entire home screen layout from a source user to one or more target users. Useful for keeping multiple profiles identical. Runs independently from the tag and collection sync.
sync
Automatic & Manual — Both Manage and Copy & Sync operations can run automatically as part of the scheduled task, or be triggered manually using the Run button → Sync home screen only.
settings Settings & Utilities
key
Trakt Client ID — Required for External sources pointing to Trakt lists. Create a new app at trakt.tv/oauth/applications, set Redirect URI to http://localhost, and copy the Client ID into the Settings tab.
vpn_key
MDBList API Key — Required for MDBList sources. Obtain it from mdblist.com/preferences under "API Access".
bug_report
Extended Log Output — Enables verbose logging for all sync operations. View logs using the Logs button in the top bar. Useful for diagnosing sync issues.
science
Dry Run Mode — Simulates a full sync without making any changes to the Emby database. Use it to preview which items would be tagged, removed, or collected before committing.
save
Backup & Restore — Export your entire plugin configuration to a JSON file, or restore from a previous backup. Found in the Settings tab under Backup & Restore.
play_circle
Run button options — The Run button offers three choices: Full sync (tags, collections, and home screen), Sync home screen only, and Sync tags and collections only.

add_circle_outline
Each rule compares one property against a value. Add multiple rules to a group to combine conditions.
do_not_disturb_on
The NOT button on each rule negates it — e.g. NOT Genre: Action matches items that are not action.

AND / OR within a group — the radio buttons at the top of a group control how rules in that group are combined.

AND / OR between groups — the connector badge between groups controls how groups relate to each other.
